# Content Guidelines for InsightHub.ink

This document outlines the content creation guidelines, SEO requirements, and best practices for blog posts and pages on InsightHub.ink.

## Content Strategy Overview

### Primary Focus: Tools & Developer Resources
- Tutorials on how to use our free developer tools
- Comparison guides (Tool X vs Alternatives)
- Best practices for developers
- Tool category guides

### Secondary Focus: SMM & Creator Growth
- YouTube growth strategies
- Instagram growth guides
- TikTok growth tips
- How to buy social media engagement safely
- Platform-specific guides

### Tertiary Focus: FinTrack & Personal Finance
- Budgeting for IT professionals
- Goal-based saving guides
- Expense tracking best practices
- Finance management tips

## Content Requirements

### Minimum Length
- **1500-2500 words** per blog post
- Longer, comprehensive guides perform better in search
- Aim for depth over breadth

### Structure Requirements

#### H1/H2/H3 Hierarchy
- **One H1 per page** - Main keyword-focused title
- **H2 sections** - One main keyword per section
- **H3 subsections** - Supporting topics
- Example structure:
  ```
  H1: How to Use JSON Formatter: Complete Guide 2025
  H2: What is a JSON Formatter?
  H2: Why Use Our JSON Formatter Tool?
  H2: Step-by-Step Guide to Formatting JSON
    H3: Formatting Options
    H3: Validation Features
  H2: Common JSON Formatting Issues
  H2: Best Practices for JSON Formatting
  ```

### Keyword Research Process

1. **Primary Keyword**: Main topic (e.g., "JSON formatter")
2. **Long-tail Keywords**: Specific queries (e.g., "how to format JSON online")
3. **Related Keywords**: Semantic variations (e.g., "JSON validator", "JSON beautifier")
4. **Intent-based Keywords**: User intent (e.g., "best JSON formatter", "JSON formatter tool")

### Internal Linking Rules

#### Anchor Text Guidelines
- Use descriptive anchor text (NOT "click here" or "read more")
- Match anchor text to target page's main keyword
- Examples:
  - ✅ "Use our [JSON formatter tool](/tools/json/formatter) to format your data"
  - ✅ "Explore our [social media growth services](/smm) for instant results"
  - ✅ "Try [FinTrack](/fintrack) to track your expenses"
  - ❌ "Click here to see our tools"
  - ❌ "Read more about this"

#### Linking Frequency
- **Minimum 3-5 internal links** per blog post
- Link to relevant:
  - Tool pages (for tool-related posts)
  - `/smm` page (for SMM-related posts)
  - `/fintrack` page (for finance-related posts)
  - Related blog posts
  - Category pages

#### Contextual Linking
- Add links naturally within content
- Place links where they add value
- Use CTA sections for prominent product links

### Image Requirements

#### Image Specifications
- **Format**: WebP or optimized JPG/PNG
- **Size**: Maximum 1200px width for featured images
- **Alt Text**: Descriptive, keyword-rich alt text
  - ✅ "JSON formatter tool interface showing formatted code"
  - ❌ "image1.jpg" or "screenshot"

#### Image Placement
- Featured image at top of post
- Screenshots for tutorials (every 300-400 words)
- Infographics for complex topics
- Charts/graphs for data-driven content

### Meta Descriptions

#### Requirements
- **Length**: 150-160 characters
- **Include**: Primary keyword, value proposition, CTA
- **Format**: 
  ```
  [What] + [How] + [Benefit] + [CTA]
  Example: "Learn how to format JSON online with our free JSON formatter tool. Validate, beautify, and minify JSON instantly. Try it now!"
  ```

### Title Tags

#### Requirements
- **Length**: 50-60 characters
- **Format**: Primary Keyword | Brand Name
- **Include**: Main keyword, year (if relevant), value proposition
- Examples:
  - "How to Format JSON Online: Complete Guide 2025 | InsightHub"
  - "Best JSON Formatter Tools: Comparison Guide | InsightHub"
  - "How to Get 4000 YouTube Watch Hours: Complete Guide | InsightHub"

## Content Templates

### Tool Tutorial Template

```markdown
# How to Use [Tool Name]: Complete Guide 2025

## Introduction
Brief overview of the tool and what it does.

## What is [Tool Name]?
Explanation of the tool's purpose and use cases.

## Why Use Our [Tool Name] Tool?
Benefits and features of our specific tool.

## Step-by-Step Guide
### Step 1: [Action]
Detailed instructions with screenshots.

### Step 2: [Action]
More detailed instructions.

## Common Use Cases
Real-world examples and scenarios.

## Tips and Best Practices
Expert advice for optimal usage.

## Related Tools
Links to related tools on our platform.

## Conclusion
Summary and CTA to try the tool.
```

### SMM Guide Template

```markdown
# How to [Action]: Complete Guide 2025

## Introduction
Overview of the topic and why it matters.

## Understanding [Topic]
Background information and context.

## Step-by-Step Process
Detailed instructions.

## Best Practices
Expert tips and recommendations.

## Common Mistakes to Avoid
What NOT to do.

## How Our SMM Services Can Help
Link to /smm with contextual anchor text.

## Conclusion
Summary and next steps.
```

### Comparison Post Template

```markdown
# [Tool/Service] vs Alternatives: Comparison Guide 2025

## Introduction
Why compare these options.

## [Option 1] Overview
Features, pros, cons.

## [Option 2] Overview
Features, pros, cons.

## Side-by-Side Comparison
Table or detailed comparison.

## Which Should You Choose?
Recommendations based on use case.

## Try Our [Tool/Service]
Link to our offering.

## Conclusion
Final thoughts.
```

## SEO Checklist for Each Post

### Pre-Publication Checklist
- [ ] Title tag optimized (50-60 chars, includes keyword)
- [ ] Meta description written (150-160 chars, includes keyword)
- [ ] H1 includes primary keyword
- [ ] H2 sections include related keywords
- [ ] Minimum 1500 words
- [ ] At least 3-5 internal links with descriptive anchor text
- [ ] Featured image with descriptive alt text
- [ ] Additional images with alt text (if applicable)
- [ ] Schema markup included (Article schema)
- [ ] Canonical URL set
- [ ] Open Graph tags configured
- [ ] Twitter Card configured
- [ ] CTA section added (SMM, FinTrack, or Tools)
- [ ] Related tools section (if tool-related)
- [ ] Related posts section populated

### Post-Publication Checklist
- [ ] Submit to Google Search Console
- [ ] Share on social media
- [ ] Add to internal linking from other posts
- [ ] Monitor performance in analytics

## Quality Standards

### Writing Quality
- Clear, concise, and engaging
- Free of grammatical errors
- Well-structured with proper headings
- Includes examples and screenshots
- Actionable advice and tips

### Originality
- Original content (no plagiarism)
- Unique perspectives and insights
- Real examples and case studies
- Personal experience when relevant

### Value Proposition
- Solves a real problem
- Provides actionable steps
- Includes expert insights
- Offers tools/resources

## Performance Tracking

### Metrics to Monitor
- Organic traffic from search
- Keyword rankings
- Time on page
- Bounce rate
- Internal link clicks
- Conversion rate (tool usage, SMM signups, FinTrack signups)

### Optimization Based on Data
- Update underperforming posts
- Expand on high-performing topics
- Improve internal linking based on user behavior
- A/B test CTAs and headlines
